# challenge-classification

### The objective of this project is to compare different machine learning models.  
<br/>  

## Folder Hierarchy
```
challenge-classification
|   main.py : the entry point of the program
|   README.md
|   
+---data
|       UCI_Credit_Card.csv : the dataset to study
|       
+---notebook : Few notebook to test the code
|       
\---utils
    |   cleaning.py : Called by main.py. Clean the dataset
    |   learning.py : Called by main.py. Create some model to test
    |   preprocessing.py : Called by learning.py. Display the correlation matrix
```  

## Usage
Just run `main.py` file
